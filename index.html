适配：Cocos 单文件 / ZIP / inline data:image
Workflow
Step 1. 上传单文件 H5（.html）
Step 2. 解析资源，勾选需要替换的图片
Step 3. 填写目标 App Store 链接 自动覆盖所有原链接
Step 4. 生成新的 playable HTML (记得要检查)
Notes
· 工具仅在浏览器本地运行，不会上传 playable 或素材；更新时请留意飞书通知和版本号
· 当前支持识别 window.__zip、window.__res、__adapter_zip__ 和部分 inline data:image 场景
· ZIP 内图片替换后会自动重新压缩回写，尽量避免生成文件体积异常变大
· adapter_zip 模式通过运行时 override 生效；如素材未即时变化，建议重新打开页面验证
· 工具会统一重写常见商店链接，包括 itunes.apple.com、apps.apple.com、部分内嵌 ZIP 内链接和 mraid / window.open 兜底逻辑
· 生成后的 HTML 仍建议手动检查：页面展示、首屏素材、点击跳转、最终文件体积是否正常
· 使用过程中遇到任何问题，请在飞书找子薇协助处理
当前为本地工具，仅在浏览器本地读取 / 重写文件，不会上传广告素材。
