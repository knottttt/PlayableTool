# Changelog

## 3.1.0 - 2026-03-16

### 版本概览
本次 3.1.0 重点补强了 `window.__zip` 回写后的交付稳定性，解决换图或改链接后文件体积明显变大的问题。

### 更新内容
- 新增 `__zip` 高压缩回写，重打包时显式使用 `DEFLATE`。
- 修复部分 playable 在改动后 `__zip` 压缩率下降，导致 HTML 从 4MB 膨胀到 7MB 左右的问题。
- 保持现有 `window.__zip`、`window.__res`、批量换图与跳转重写逻辑不变。

## 2.0.0 - 2026-03-13

### 版本概览
本次 2.0.0 主要围绕高频调试同一个 playable 的使用场景进行了优化，重点提升稳定性和换图效率。

### 更新内容
- 修复了连续生成 HTML 后文件体积不断变大的问题。
- 修复了反复调试时修改内容层层叠加、结果越来越不稳定的问题。
- 新增批量导入图片功能，支持一次选择多张图片自动匹配替换。
- 新增批量替换结果提示，可直接查看已匹配和未匹配数量。
- 对不适合批量替换的模式增加了明确提示，减少误操作。

### 使用体验提升
- 更适合反复导入、反复改图、反复生成的工作流程。
- 多图替换效率更高，减少逐张上传的重复操作。
- 生成结果更干净，历史残留更少，更方便检查和交付。
